# StudyAgent 开发任务拆解

## 1. 项目开发目标

开发一个基于 LLM + RAG + Agent 的智能期末复习助手。

最终实现：

-   用户上传课程 PPT
-   AI 自动解析课程内容
-   自动生成复习笔记
-   建立课程知识库
-   AI 根据课程内容回答问题
-   自动生成复习计划
-   模拟考试和错题分析

------------------------------------------------------------------------
