# StudyAgent：智能期末复习 Agent 开发方案

## 1. 项目概述

### 项目名称

StudyAgent：基于大模型的智能期末复习助手

### 项目目标

开发一个具有可视化界面的 AI Agent，帮助学生完成：

-   上传教师课程 PPT
-   自动解析课程内容
-   生成结构化复习笔记
-   提取重点知识和高频考点
-   提供智能问答
-   根据考试时间生成个性化复习计划

核心技术：

-   Large Language Model（LLM）
-   Retrieval Augmented Generation（RAG）
-   Agent Workflow
-   Multimodal Document Understanding
-   Knowledge Base

# 3. 功能设计

## 3.1 PPT上传与解析

### 输入

用户上传：

-   pptx文件
-   pdf文件

### 处理流程

    PPT
     |
    文本提取
     |
    图片/OCR解析
     |
    内容切片
     |
    Embedding
     |
    向量数据库

### 输出

生成：

-   章节结构
-   知识点列表
-   重点标记
-   页码索引

示例：

    章节：
    Cache Memory

    核心知识：
    1. Cache映射方式
    2. 地址转换
    3. 命中率计算

    重要程度：
    ★★★★★

------------------------------------------------------------------------

# 4. 知识库建设（RAG）

## 数据处理

步骤：

1.  文档解析
2.  文本切片
3.  向量化
4.  存入数据库

推荐组件：

  功能         技术
  ------------ ------------------------
  文本解析     python-pptx
  OCR          PaddleOCR
  Embedding    OpenAI Embedding / BGE

# 5. Agent设计

## 5.1 Document Agent

职责：

-   解析课程资料
-   建立知识结构
-   标记重点内容

输入：

    课程PPT

输出：

    课程知识地图

------------------------------------------------------------------------

## 5.2 Tutor Agent

职责：

-   回答课程问题
-   基于PPT内容解释
-   提供考试角度分析

示例：

用户：

> Cache为什么提高CPU性能？

AI：

根据课程PPT：

-   局部性原理
-   命中率提升
-   减少访问主存次数

考试可能考：

解释Cache工作机制。

------------------------------------------------------------------------

## 5.3 Planner Agent

输入：

-   考试日期
-   每日学习时间
-   当前掌握程度

输出：

个性化学习计划。

示例：

    Day1:
    CPU结构

    Day2:
    存储系统

    Day3:
    IO系统

    Day4:
    刷题

    Day5:
    复盘

------------------------------------------------------------------------

## 5.4 Examiner Agent

功能：

自动生成：

-   模拟题
-   简答题
-   选择题
-   易错题

并进行：

-   自动评分
-   错误分析
