# StudyAgent 技术实现文档

## 1. 技术目标

实现一个基于 LLM + RAG + Agent 的智能期末复习系统。

核心能力：

-   课程资料解析
-   知识库构建
-   智能问答
-   学习计划生成
-   模拟考试
-   学习状态分析

------------------------------------------------------------------------

# 4. PPT解析模块

## 输入

    course.pptx

## 流程

    PPT
     |
    python-pptx
     |
    文本提取
     |
    OCR图片识别
     |
    文本清洗
     |
    Chunk切分
     |
    Embedding
     |
    Vector DB

------------------------------------------------------------------------

## 输出格式

``` json
{
"title":"Cache Memory",
"content":"Cache利用局部性原理提高访问速度",
"page":15,
"importance":5
}
```

# 5. RAG实现流程

## 文档切片

原则：

-   一个知识点一个chunk
-   保留章节信息
-   保留页码

Example:

    Chunk:

    课程：
    计算机组成

    章节：
    存储系统

    知识点：
    Cache映射

------------------------------------------------------------------------

## 检索流程

用户问题：

    Cache如何提高性能？

↓

Embedding

↓

向量搜索

↓

召回相关PPT内容

↓

LLM生成回答

------------------------------------------------------------------------

# 6. Agent设计

## Document Agent

Prompt:

    你是课程资料分析专家。

    任务：
    分析上传PPT，
    提取：
    1.章节结构
    2.核心知识点
    3.考试重点
    4.难点

------------------------------------------------------------------------

## Tutor Agent

功能：

-   课程问答
-   概念解释
-   考试角度分析

输入：

    问题 + 检索内容

输出：

    解释
    +
    对应PPT章节
    +
    考试建议

------------------------------------------------------------------------

## Planner Agent

输入：

    考试日期
    剩余时间
    学习水平
    每天时间

输出：

    Day1:
    学习章节

    Day2:
    练习题

    Day3:
    复习

------------------------------------------------------------------------

## Examiner Agent

功能：

生成：

-   选择题
-   判断题
-   简答题

自动：

-   批改
-   分析错误原因

------------------------------------------------------------------------

# 7. API设计

    /upload

    file

    "status":"success",

POST

}
